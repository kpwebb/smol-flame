## 🕯️ smol-flame

A tiny template for minimalism on the edge. Maximum legibility, zero magic.

Combines:
- Hono
- Tailwind + PostCSS
- Bun (dev)
- Cloudflare Workers (deploy)

To use: 

```$ bun run make```

To build (for Cloudflare Pages support):

```$ bun run bake```

To deploy (for Cloudflare Workers support):

```$ bun run shake```
